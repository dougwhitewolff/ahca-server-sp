**If Person Doesn\'t Answer**

**Process:**

1.  Agent comes back on: \"It looks like \[Name\] isn\'t available right
    now. Can I get your name and number so they can call you back?\"

2.  Send text notification to:

    - April (always)

    - The person who missed the call

3.  Text includes caller\'s name, number, and reason for calling

**Routing Method**

**Open-ended question approach** (not menu-based):

1.  Ask \"What can I help you with today?\"

2.  Agent determines intent from natural response

3.  If unclear, offer list of options: \"You might want to talk to
    someone about volunteering, deliveries, donations\...\"

4.  Forward to appropriate person
